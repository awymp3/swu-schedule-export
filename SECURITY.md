# 安全说明

## 账号与本地数据

- 西南大学统一身份认证账号和密码保存在项目根目录的 `.env`，该文件已在 `.gitignore` 中排除。
- `captured.js` 可能包含课程、教师和上课地点信息，同样不会提交到 Git。
- 请不要将 `.env`、`captured.js`、`drivers/` 或 `.runtime/` 上传到公开仓库、网盘或聊天群。

## 使用边界

本项目仅用于导出本人在西南大学教务系统中可正常查看的课程安排。使用者应遵守西南大学相关规定和教务系统的使用要求，不应将其用于批量查询、绕过访问控制或影响系统正常运行。

本项目与西南大学及其教务系统没有隶属、授权或合作关系。

## 漏洞反馈

请勿在公开 Issue 中粘贴统一身份认证账号、密码、Cookie、课程数据或可复现的敏感页面内容。涉及凭据泄露、数据暴露或安全绕过的问题，请先联系仓库维护者并提供脱敏后的说明。
